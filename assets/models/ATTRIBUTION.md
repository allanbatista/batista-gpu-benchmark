# Default model

`benchmark.glb` — "Astronaut in a White Suit", generated by Allan Batista with
Meshy AI (2026-07-11). This is the official bundled benchmark model; runs with
any other model are reported as `custom`.

To use your own model, replace this file or pass `--model <path>` on the
command line. The model is auto-scaled and grounded via its bounding box.
